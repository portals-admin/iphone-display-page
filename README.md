# iphone-display-page

A simple web page that displays an iPhone with visual styling and responsive design.

## Overview

This project is a static web page showcasing an iPhone mockup with clean visual styling. It is designed to be lightweight and responsive across different screen sizes.

## Usage

Open `index.html` in any modern web browser — no build step or dependencies required.

## Features

- iPhone visual mockup
- Responsive design
- Pure HTML/CSS, no frameworks
